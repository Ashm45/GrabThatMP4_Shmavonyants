\# Мини-ТЗ: GrabThatMP4



\## 1. Название

GrabThatMP4 (Скачивание MP4 по ссылке)



\## 2. Цель

Программа помогает сохранять видео из различных соцсетей в виде файла на компьютер пользователя по ссылке



\## 3. Пользователь

Обычный пользователь ПК (студент)

Создатель контента



\## 4. Функциональные требования (FR)

* FR1: Добавить ссылку на видео
* FR2: Показать скачанные видео
* Изменить качество/формат/название видео
* Удалить видео



\## 5. Нефункциональные требования (NFR)

* Приложение запускается  без ошибок
* Понятные сообщения об ошибках
* Данные будут храниться локально



\## 6. Модель данных (черновик)

\*\*Видео Video\*\*

* Id (число)
* Title(строка, не пустая)
* Качество(144p, 360p, 720p, 1080p, 1440p, 2K, 4K)
